# SnapBeam Assets Directory

Place the following assets in this directory:

## Required Assets

### App Icons
- `app_icon.png` - 1024x1024 PNG (main app icon)
- `app_icon_foreground.png` - 512x512 PNG with transparent background (for adaptive icons)
- `splash_logo.png` - 288x288 PNG (splash screen logo)
- `splash_logo_android12.png` - 240x240 PNG (Android 12+ splash)
- `splash_background.png` - 2880x1800 PNG (splash background gradient)
- `splash_branding.png` - Optional branding image

### Font Files
Create `fonts/` subdirectory and add:
- `Poppins-Regular.ttf`
- `Poppins-Medium.ttf`
- `Poppins-SemiBold.ttf`
- `Poppins-Bold.ttf`

Download from: https://fonts.google.com/specimen/Poppins

## Design Guidelines

### App Icon
- Background: Gradient from #6366F1 to #EC4899
- Icon: White camera icon in center
- No text on icon
- Rounded corners (handled automatically)

### Colors
- Primary: #6366F1 (Indigo)
- Secondary: #EC4899 (Pink)
- Tertiary: #14B8A6 (Teal)
- Background: #FFFFFF (Light) / #1F2937 (Dark)

### Screenshots for Stores
Required sizes:
- iPhone 6.7": 1290 x 2796
- iPhone 6.5": 1242 x 2688
- iPhone 5.5": 1242 x 2208
- Android Phone: 1080 x 1920
