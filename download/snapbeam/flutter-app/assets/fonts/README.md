# Fonts Directory

Add the Poppins font files here:

1. Download Poppins from Google Fonts:
   https://fonts.google.com/specimen/Poppins

2. Extract and copy these files:
   - Poppins-Regular.ttf
   - Poppins-Medium.ttf
   - Poppins-SemiBold.ttf
   - Poppins-Bold.ttf

3. Place them in this directory:
   flutter-app/assets/fonts/
