# SnapBeam

Send moments. Instantly live on your loved one's home screen.

## Features

- **No Accounts Required** - Just use 8-character connection codes
- **Home Screen Widgets** - Photos appear instantly on Android/iOS widgets
- **Daily Free Sends** - 2 photos per day for free users
- **Premium Coming Soon** - Unlimited sends, HD quality, custom themes

## Download

Download the latest APK from [Releases](../../releases).

## Build

```bash
# Get dependencies
flutter pub get

# Build debug APK
flutter build apk --debug

# Build release APK
flutter build apk --release

# Build release AAB (for Play Store)
flutter build appbundle --release
```

## Package ID

- Android: `app.snapbeam.photo`
- iOS: `app.snapbeam.photo`
